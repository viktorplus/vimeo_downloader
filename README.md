# Загрузчик видеоуроков с Vimeo

Скрипт автоматически скачивает видеоуроки с Vimeo, включая приватные ссылки с токеном (вида `vimeo.com/ID/токен?fl=pl&fe=cm`). Заменяет ручное использование сторонних сервисов типа catch.tube.

## Требования

- Python 3.10+
- ffmpeg (для объединения видео и аудио)

### Установка ffmpeg (Windows)

```powershell
winget install ffmpeg
```

или скачать вручную с <https://ffmpeg.org/download.html> и добавить в PATH.

### Установка зависимостей Python

```bash
pip install -r requirements.txt
```

Или вручную:

```bash
pip install flask yt-dlp selenium curl-cffi
```

## Использование

### Локальное меню с кнопками (рекомендуется)

Запуск:

```bat
start_lessons_menu.bat
```

Откроется локальный интерфейс: `http://127.0.0.1:5050`.

Что умеет меню:

- список всех уроков из `lessons_list.txt`
- фильтр по предмету и поиск по названию
- выбор качества для каждого урока
- кнопка `Скачать` у каждой записи
- кнопка `Обновить список из LMS` (добавляет новые уроки без дублей)

### Обновить список, когда в кабинете появились новые записи

Вариант 1 (прямо в меню, без перезапуска):

1. В `http://127.0.0.1:5050` нажмите кнопку `Обновить список из LMS`.
2. Откроется браузер LMS.
3. Войдите в кабинет, если потребуется.
4. Дождитесь сообщения о завершении и авто-обновления страницы.

Вариант 2 (отдельным скриптом):

Запуск:

```bat
update_lessons_list.bat
```

Как это работает:

1. Откроется браузер на странице записей LMS.
2. Если нужно, войдите в кабинет.
3. Нажмите `Enter` в консоли.
4. Скрипт вытащит уроки из таблицы и допишет в `lessons_list.txt` только новые записи (без дублей по Vimeo URL).

После обновления через bat-файл перезапустите `start_lessons_menu.bat`, чтобы увидеть новые уроки в меню.

Куда сохраняются файлы:

- по умолчанию в `videos/<ПРЕДМЕТ>/`
- пример: `videos/Python Fundamentals/`
- папку можно изменить в верхнем поле интерфейса перед запуском скачивания

### Скачать одно видео

```bash
python download_lessons.py "https://vimeo.com/1177373233/a17469db60?fl=pl&fe=cm"
```

Видео сохранится в папку `./videos/` с именем по заголовку урока.

### Указать качество

```bash
python download_lessons.py "https://vimeo.com/..." --quality 720
```

Доступные значения: `360`, `480`, `720`, `1080`, `1440`, `2160`  
По умолчанию: `1080`

### Указать папку сохранения

```bash
python download_lessons.py "https://vimeo.com/..." --output "D:/Уроки/Python"
```

### Скачать список уроков из файла

Создать текстовый файл `urls.txt` — по одной ссылке на строку.  
Строки, начинающиеся с `#`, игнорируются (можно добавлять комментарии).

```text
# Python курс
https://vimeo.com/111111111/aabbccdd?fl=pl&fe=cm
https://vimeo.com/222222222/eeff0011?fl=pl&fe=cm

# QA курс
https://vimeo.com/333333333/112233aa?fl=pl&fe=cm
```

Запуск:

```bash
python download_lessons.py --file urls.txt --quality 1080 --output "D:/Уроки"
```

### Посмотреть доступные форматы без скачивания

```bash
python download_lessons.py "https://vimeo.com/..." --list-formats
```

## Все параметры

| Параметр               | Короткий | Описание                                              |
|------------------------|----------|-------------------------------------------------------|
| `url`                  |          | URL видео (позиционный аргумент)                      |
| `--file ФАЙЛ`          | `-f`     | Текстовый файл со списком URL                         |
| `--quality КАЧЕСТВО`   | `-q`     | Максимальное разрешение (360/480/720/1080/1440/2160)  |
| `--output ПАПКА`       | `-o`     | Папка для сохранения файлов                           |
| `--list-formats`       | `-l`     | Показать форматы без скачивания                       |

## Структура файлов

```text

├── download_lessons.py   # основной скрипт
```

Скачанные видео по умолчанию сохраняются в:

```text
vimeo_downloader/
└── videos/
    ├── Python 88 Онбординг в проект Python.mp4
    ├── Database 59 Урок.mp4
    └── ...
```

## Часто задаваемые вопросы

**Почему скрипт, а не catch.tube?**  
Ручной процесс: открыть сайт → вставить ссылку → выбрать качество → скачать.  
Со скриптом: положить все ссылки в `urls.txt` и запустить один раз.

**Работают ли приватные ссылки с токеном?**  
Да. Ссылки вида `vimeo.com/ID/токен?fl=pl&fe=cm` работают напрямую.

**Что делать если ошибка "ffmpeg not found"?**  
Установить ffmpeg (см. выше). Без ffmpeg видео скачается, но без аудио или в другом формате.

**Можно ли скачать в формате, отличном от mp4?**  
Скрипт принудительно объединяет в mp4. Для изменения — отредактировать параметр `merge_output_format` в функции `download()`.
